# QR Code Service

Сервис генерирует и хранит QR-код и основную информацию, полученную из сообщений Kafka. 

Вместо реального QR кода генерируем хеш по алгоритму MD5 от полученных из Kafka данных. Далее под QR кодом подразумеваем именно этот хеш.

## REST API:
* GET ```/qr/{passport}```: выдает самый свежий код пользователю для его последующего предъявления по паспорту;
* GET ```/qr/check?сode={code}```: удостоверяет подлинность и актуальность QR кода (тем самым подтверждает полномочия владельца);

## Дополнительные требования:
* На каждое уникальное событие вакцинации должен быть создан ровно один QR код.
* Сервис обеспечивает персистентное хранение QR кодов в БД.
* Сервис зарегистрирован в service registry

Сборка JAR: ```mvn clean compile assembly:single```

Технический стек: Spring Boot, Spring MVC, Spring Kafka
